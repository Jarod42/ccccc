NAME

llvm-stress - generate random .ll files

SYNOPSIS

llvm-cov [-gcno=filename] [-gcda=filename] [dump]

DESCRIPTION

The llvm-stress tool is used to generate random .ll files that can be used to test different components of LLVM.

OPTIONS

-o filename

Specify the output filename.

-size size

Specify the size of the generated .ll file.

-seed seed

Specify the seed to be used for the randomly generated instructions.

EXIT STATUS

llvm-stress returns 0.

AUTHOR

llvm-stress is maintained by the LLVM Team (http://llvm.org/).
